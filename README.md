# 🤖 RoboVision AI

**Real-Time AI Perception System for Humanoid Robots**

RoboVision AI is an open-source modular perception system designed for humanoid robots. It combines computer vision, deep learning, and robotics frameworks to enable robots to perceive, understand, and interact with the real world in real time.

The project is built using **Python**, **OpenCV**, **YOLO**, **MediaPipe**, **PyTorch**, and **ROS2**, with a focus on Human-Robot Interaction (HRI).

---

## 🚀 Vision

The goal of RoboVision AI is to create a production-ready perception stack that can be integrated into humanoid robots for applications such as:

* Human detection
* Face recognition
* Hand gesture recognition
* Pose estimation
* Emotion recognition
* Person tracking
* Voice interaction
* Robot command generation
